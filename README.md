# CoronaKvach - COVID-19 Tracker

An Android application for tracking COVID-19 statistics with real-time data updates and an intuitive dashboard.

## Features

- Real-time COVID-19 statistics
- Country-wise and global data
- Interactive charts and graphs
- Daily case updates
- Clean and informative UI

## Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM
- **Networking:** Retrofit
- **Charts:** MPAndroidChart
- **UI:** Material Design

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/bhartendukodes/CoronaKvach.git
```
2. Open in Android Studio
3. Build and run
